One mental leap that I had to make is the difference between "timeline" (or "news feed") feeds and regular old "profile" (or "user") feeds. They're all feeds, but "timeline" feeds follow "profile" feeds - "profile" feeds do not follow other "profile" feeds.

Each "user" should have 2 feeds: their own "profile" feed (which is just the activities that they post to their profile) and a separate "timeline" feed (which follows other user's "profile" feeds).

BB-8's "Timeline"

(where BB-8 can see status updates from all the other users he "follows")

Feed type: Flat

Feed group: "timeline" (this can be named whatever you want - create feed groups in the GetStream.io dashboard)

This feed follows: nobody yet! (only flat feeds can be followed)

BB-8's "Profile"

(where anyone can see status updates from BB-8)

Feed type: Flat

Feed group: "profile" (this can be named whatever you want - create feed groups in the GetStream.io dashboard)

This feed follows: nobody, and it shouldn't follow anyone! It's BB-8's personal feed.

Finn's "Profile"

(where anyone can see status updates from Finn)

Feed type: Flat

Feed group: "profile" (this can be named whatever you want - create feed groups in the GetStream.io dashboard)

This feed follows: nobody, and it shouldn't follow anyone! It's Finn's personal feed.
